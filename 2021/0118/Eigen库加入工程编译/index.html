Eigen库加入工程编译

发表于 2021-01-18 更新于 2021-03-12 分类于 Programming
本文字数： 1.8k 阅读时长 ≈ 2 分钟

Eigen

    Eigen是一个高效的矩阵运算库。

CMake工程加入Eigen

    Eigen3按文档所说，可以直接加入CMake工程使用，按Eigen文档的CMakeLists.txt编写方式，CMakeLists.txt、example.cpp和Eigen库文件夹放在同一目录下

cmake_minimum_required (VERSION 3.0)
project (myproject)
find_package (Eigen3 3.3 REQUIRED NO_MODULE)
add_executable (example main.cpp)
target_link_libraries (example Eigen3::Eigen)

    CMake生成报错，找不到Eigen3Config.cmake或者eigen3-config.cmake，实际上搜索Eigen文件夹，确实没有这两个文件，仔细看报错信息，是说找不到Eigen3的库，解决方法是 CMAKE_PREFIX_PATH 或者 Eigen3_DIR 个变量指向这两个文件的目录，但实际上找不到这两个目录。其实下载下来的Eigen代码本来就是CMake工程，它也应该可以CMake一下。用CMake-gui，修改 CMAKE_INSTALL_PREFIX 的路径，它是生成的文件所在目录，去该目录，发现有Eigen3Config.cmake文件生成，执行 make install，最终生成的include和share文件夹就是我们需要的，其他的全部不要了，修改CMakeLists.txt如下，只需要多加一行就行

cmake_minimum_required (VERSION 3.0)
project (myproject)

# 设置Eigen3Config.cmake文件所在目录
set(Eigen3_DIR "Eigen3/share/eigen3/cmake")
# 加入头文件
include_directories("libs/Eigen3/include" "libs/Eigen3/include/eigen3")

find_package (Eigen3 3.3 REQUIRED NO_MODULE)
 
add_executable (example main.cpp)
target_link_libraries (example Eigen3::Eigen)

    最终结果，CMake生成通过，编译通过，可以正常运行。Eigen想加入CMake工程，还是需要靠Eigen3Config.cmake这个文件，里面有很多路径的设置

OpenCV加入Eigen编译

    有两种方式，能读出版本号的都是可以的方法

  1. 直接按照上面加入CMake工程的，Eigen3_DIR 指向 Eigen3Config.cmake 这个文件的目录就行
  2. 直接用源码，EIGEN_INCLUDE_PATH 指向源码跟目录就行
